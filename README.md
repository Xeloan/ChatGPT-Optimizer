# ChatGPT 折叠优化插件

一个用来缓解 ChatGPT 长对话卡顿的 Chrome 扩展。正常情况下，ChatGPT网页版在长对话以后会非常卡，表现为输个字都会一个个慢慢蹦出来。
核心思路是把屏幕外的历史消息隐藏掉，在需要的时候再恢复，从而减少页面渲染压力。

---

## 功能

* 长对话场景下明显更流畅
* 屏幕内的消息自动渲染
* 滚出视野的旧消息自动隐藏
* 不影响浏览器搜索（Ctrl + F3）
* 不影响对话上下文

---

## 注意

ChatGPT 在对话变长之后变卡，主要原因是页面中累积了大量 DOM 节点，而不是模型本身变慢。网络或者模型性能问题导致的卡顿依然存在（但是你打字肯定不会卡），然后刚进长对话会稍微顿一下因为对话隐藏也需要时间，但是加载对话时间长纯粹是OpenAI土豆服务器的问题，无解。

---

## 安装

开发模式安装：

1. 打开 Chrome
2. 进入 `chrome://extensions/`
3. 打开右上角开发者模式
4. 点击加载已解压的扩展程序
5. 选择项目目录（解压好的文件夹，你找个地方存下）

安装完成后刷新 ChatGPT 页面即可生效。

Firefox方法类似，请自行谷歌，反正就是开发者模式加载扩展。
