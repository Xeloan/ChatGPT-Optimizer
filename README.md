# ChatGPT 折叠优化插件

一个用来缓解 ChatGPT 长对话卡顿的 Chrome 扩展。正常情况下，ChatGPT网页版在长对话以后会非常卡，表现为输个字都会一个个慢慢蹦出来。
核心思路是把屏幕外的历史消息隐藏掉，在需要的时候再恢复，从而减少页面渲染压力。

---

## 功能

* 长对话场景下明显更流畅
* 屏幕内的消息自动渲染
* 滚出视野的旧消息自动隐藏
* 不影响浏览器搜索（Ctrl + F3）
* 不影响对话上下文

---

## 注意

ChatGPT 在对话变长之后变卡，主要原因是页面中累积了大量 DOM 节点，而不是模型本身变慢。如果因为网络或者模型性能问题导致的卡顿依然存在（但是你打字肯定不会卡），然后刚进长对话会稍微顿一下因为对话隐藏也需要时间。

---

## 安装

开发模式安装：

1. 打开 Chrome
2. 进入 `chrome://extensions/`
3. 打开「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目目录

安装完成后刷新 ChatGPT 页面即可生效。

---

## 使用说明

* 默认保留最近一部分消息（例如 20 条）
* 更早的消息会根据是否在视野内自动折叠
* 滚动页面时会自动恢复可见区域的消息

---

## 注意事项

* 本插件只修改页面渲染，不会：

  * 修改用户输入
  * 修改发送给模型的上下文
  * 上传或存储任何数据

* ChatGPT 页面结构如果发生变化，可能需要调整选择器

## 核心实现

* `getMessages()`：获取消息节点
* `collapseMessage()`：折叠消息
* `restoreMessage()`：恢复消息
* `IntersectionObserver`：控制可见区域的展开/折叠
* `MutationObserver`：监听新消息并动态处理
